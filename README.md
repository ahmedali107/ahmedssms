# Ahmed's Student Management System (SMS)

A clean and modern **Student Management System** web application built with **HTML, CSS, and Bootstrap**.  
It allows administrators to manage students, courses, and records efficiently.

---

## Features

- **Student Directory**  
  View all students with details like name, email, age, and enrolled course.

- **Add/Edit/Delete Students**  
  Easily manage student records with intuitive forms and static confirmation modals.

- **Course Management**  
  Add and manage courses with course codes and badges.

- **Responsive Design**  
  Fully responsive UI for desktops, tablets, and mobile devices.

- **Professional UI**  
  Modern and attractive interface using **Bootstrap 5** and custom CSS.

---

## Tech Stack

- **Frontend:** HTML, CSS, Bootstrap 5  
- **Icons:** Bootstrap Icons  
- **Fonts:** Google Fonts (Inter)
